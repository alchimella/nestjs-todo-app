## Description

Todo rest API NestJS.

## Installation

```bash
$ npm install
```

## Run mongo

```
sh scripts/run_mongo.sh
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run start:prod
```

## Swagger

Swagger use basic auth. 
Configuring in .env file. See in .env.example

Swagger default path:
http://localhost:3000/api/docs/
